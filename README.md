# Student Course Registration System (Java Project)

## 📌 Overview
This is a console-based Java application that manages student course registration efficiently. The system allows administrators to manage courses and students to register for available courses using a simple menu-driven interface.

---

## 🚀 Features
- Add new courses  
- Display all available courses  
- Search courses by ID or name  
- Update course information  
- Delete courses  
- Student course registration  
- View registered courses  
- Simple and user-friendly console interface  

---

## 🛠️ Technologies Used
- Java  
- Object-Oriented Programming (OOP)  
- ArrayList (Collections Framework)  
- Exception Handling  
- File Handling (if used)  

---

## 🎯 Project Objective
The main goal of this project is to automate the course registration process and demonstrate the practical use of Java OOP concepts such as:
- Encapsulation  
- Classes and Objects  
- Polymorphism (if implemented)  
- Inheritance (if implemented)  

---

## ▶️ How to Run
1. Clone this repository  
2. Open the project in any Java IDE (IntelliJ / Eclipse / NetBeans / VS Code)  
3. Compile and run the `Main.java` file  
4. Follow the menu options in the console
